Structure

  • 이 저장소는 Turborepo로 구성된 모노레포입니다. 모든 앱 및 패키지 목록은 다음과 같습니다 (각 앱에는 자체 README가 있습니다).

Apps

  • apps/storefront: Next.js 전자 상거래 스토어 프론트 엔드 스타터 팩
  • apps/saleor-app-checkout: 설정 및 테마 관리 대시 보드, Checkout용 백엔드 및 Checkout 스토어프론트를 갖춘 Next.js Saleor 앱 – 확장 / 수정이 가능합니다.

Packages

  • packages/ui-kit: Storefront 및 Checkout용 UI 키트
  • packages/eslint-config-checkout: eslint 구성
  • packages/tsconfig: monorepo 전체에서 사용되는 tsconfig.json 파일
  • packages/checkout-storefront: Checkout 스토어프론트 컴포넌트
  • packages/checkout-common: 공통 유형 및 유틸리티
  • packages/env-vars: 패키지 간 환경 변수 가져 오기

Getting started

이 모노레포는 패키지 관리자로 pnpm을 사용합니다. 따라서 먼저 설치해야합니다. 설치하려면 다음을 실행하십시오.

npm install -g pnpm
  • 의존성을 설치하려면 다음을 실행하십시오.
pnpm i
  • 모든 개발 서버를 시작하려면 다음을 실행하십시오.
pnpm dev

이 명령은 서로 다른 포트에서 실행되는 4 개의 애플리케이션을 시작합니다.

  • 특정 앱을 시작하려면 다음을 실행하십시오.
pnpm dev --filter=saleor-app-checkout

Building

  • 모든 앱 및 패키지를 빌드하려면 다음 명령을 실행하십시오
pnpm run build
  • 특정 앱이나 패키지를 빌드하려면 다음 명령을 실행하십시오.
  • 이 예에서는 apps / saleor-app-checkout 만 빌드합니다
pnpm run build --filter=saleor-app-checkout

Starting production server

  • 개발을 완료한 후 storefront 및 saleor-app-checkout 애플리케이션을 프로덕션과 유사한 환경에서 시작하는 데 사용할 수있는 명령은 다음과 같습니다.
pnpm build
pnpm start
  • 또는 다음 명령 중 하나를 실행하여 특정 애플리케이션
pnpm start:saleor-app-checkout

Ports

  • apps/react-storefront: http://localhost:3000
  • apps/saleor-app-checkout: http://localhost:3001
  • packages/checkout-storefront: http://localhost:3002
  • packages/ui-kit storybook: http://localhost:6006

Other

GraphQL Schema

  • Saleor 인스턴스에서 정의된 SALEOR_API_URL 환경 변수를 기반으로 최신 스키마에서 GraphQL 코드를 생성하려면 다음 명령을 실행하십시오:
pnpm run generate

Code formatting

  • 코드를 커밋하기 전에 git pre-hooks에서 staged changes를 검사하여 코드 스타일을 따르는지 확인합니다. 코드를 직접 서식 지정하려면 다음 명령을 실행하십시오:
pnpm lint

Env variables

  • .env 파일 내에서 환경 변수를 변경하십시오
  • SALEOR_API_URL - Saleor의 GraphQL 엔드 포인트
  • 예시 http://localhost:8000/graphql/(일반적으로 docker빌드시 실행되는 끝점)

VS Code를 사용한 디버깅

  • 저장소에는 사용 준비가 완료된 VS Code 디버거 구성 (.vscode/launch.json)이 포함되어 있습니다.
  • 디버그 모드에서 서버 시작
pnpm debug

VS Code GraphQL Extension

  • VSCode용 GraphQL 확장은 GraphQL 프로젝트에 대한 구문 강조, 유효성 검사 및 정의로 이동, 호버 정보 및 gql 태그로 주석이 달린 쿼리와 같은 언어 기능을 추가합니다.

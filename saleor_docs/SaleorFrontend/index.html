Saleor Frontend

Motivation

🏎️ 현대적이고 빠릅니다: SSR, ISR 및 이미지 최적화와 같은 전자상거래에 대한 최신트렌드를 적용합니다.

💳 통합 결제: Adyen, Mollie 또는 Stripe와 같은 현대적인 결제 공급 업체와의 통합을 활용합니다(한국에서 사용 가능한 결제모듈은 만들어서 붙여야합니다).

🛠️ 쉽게 사용자 정의할 수 있습니다: TailwindCSS는 쉽게 확장하고 조정하거나 좋아하는 CSS 솔루션입니다.

👌 Out-of-the-box 작동합니다: DX를 고려하여 구축된 사전 구성된 도구입니다.

기술 Stack

  • Next.js
  • TypeScript
  • GraphQL with Apollo Client
  • Tailwind CSS
  • Turborepo
  • Saleor Checkout

Development

먼저 pnpm(성능에 중점을 둔 npm)을 전역으로 설치하세요

npm install -g pnpm

그런 다음 종속성을 설치하세요

pnpm i

서버를 시작하려면 다음을 실행하세요

pnpm dev

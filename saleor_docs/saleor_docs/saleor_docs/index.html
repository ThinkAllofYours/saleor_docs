Introduction

Saleor는 온라인 스토어를 구축하기 위한 전자상거래 플랫폼입니다. 현대적이고 풍부한 쇼핑 경험을 제공하는 모든 기능을 갖춘 멋진 클라이언트 인터페이스를 생성할 수 있습니다. 관리 대시보드를 사용하여 제품, 인력 및 기능을 쉽게 관리할 수 있습니다.Saleor는 소규모, 중소규모 및 대기업 수준의 소매업자를 위한 솔루션을 제공하며 물리적 및 디지털 재고 모두를 다룹니다.

Saleor 의 중요한 장점 중 하나는 일반적인 온라인 스토어와 PWA 애플리케이션 두 가지 형태로 제공된다는 것입니다. PWA는 고객이 스토어를 어떤 기기에 다운로드하여 오프라인으로 브라우징할 수 있도록 합니다. 고객은 온라인 쇼핑을 할 때처럼 주문 및 결제도 가능합니다.

Saleor의 오픈 소스 플랫폼은 무료로 사용할 수 있습니다. Python GraphQL API와 여러 TypeScript 앱으로 구성됩니다.

Architecture

Saleor는 세 가지 독특한 구성 요소로 구성됩니다

첫 번째는 Saleor Core로, GraphQL API를 노출하는 saleor core(백엔드 서버)입니다. 코어는 Python(django)으로 작성되어 있으며 사용자 인터페이스를 갖고 있지 않습니다. PostgreSQL 데이터베이스에서 상태를 유지하고 Redis에서 일부 정보를 캐싱합니다 .

두 번째는 상점을 운영하는 데 사용할 수 있는 Saleor 대시보드입니다. 대시보드는 브라우저에서 실행되는 React 애플리케이션으로 코어 서버와 통신합니다. 정적 웹사이트이므로 백엔드 코드가 없습니다.

마지막으로, Next.js Storefront은 React와 Next.js로 구현된 예제 스토어 프론트입니다. 기본 제공되는 Saleor SDK를 사용하여 귀하의 요구에 맞게 코드를 커스터마이징하거나 사용자 정의 스토어 프론트를 구축할 수 있습니다.

  • 모두 GraphQL로 통신합니다.

  • Saleor을 직접 수정할 수는 있지만, 배포가 업스트림 Saleor와 다르게 되면 업데이트하기 어려워지므로 권장하지 않습니다.

  • Saleor을 확장하는 가장 좋은 방법은 웹훅과 GraphQL을 사용하여 Saleor와 통신하는 외부 애플리케이션을 사용하는 것입니다.
